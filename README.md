# Local Dev
To run this project locally, follow these steps:

## Prerequisites
- Node.js (version 25.2 or higher)
- Git
## Installation

1. Clone the repository
```bash
git clone https://github.com/hiedanoajuu/hiedanoajuu.github.io.git
```

2. Install dependencies
```bash
npm install
```

3. Start the server
```bash
hexo server
```

4. Open your browser
Navigate to <http://localhost:4000>
